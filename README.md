# Portal Acessível

O **Portal Acessível** é uma plataforma web desenvolvida para apoiar estudos e práticas relacionadas à acessibilidade digital, servindo como complemento ao projeto de Iniciação Científica. O objetivo é reunir informações, recursos e boas práticas para promover o desenvolvimento de interfaces mais inclusivas e acessíveis.

## Objetivos

- Divulgar conceitos fundamentais de acessibilidade digital;
- Apresentar diretrizes e recomendações para o desenvolvimento web acessível;
- Facilitar o acesso a recursos, ferramentas e materiais educativos;
- Incentivar a criação de aplicações que atendam às necessidades de diferentes perfis de usuários.

## Tecnologias Utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router

## Funcionalidades

- Interface responsiva;
- Navegação intuitiva;
- Conteúdo organizado sobre acessibilidade;
- Componentes modernos e reutilizáveis;
- Estrutura preparada para expansão do conteúdo.
